활용 방향 구체화의 어려움

데이터를 찾더라도 분석, 시각화, 콘텐츠 제작에 어떻게 활용할지 정리하기 어렵습니다.

AI 추천의 신뢰성 문제

일반적인 AI 답변은 실제 존재하지 않는 데이터를 추천할 가능성이 있습니다.

서비스 해결 방식

이 서비스는 AI 추천과 실제 공공데이터포털 API 조회를 결합하여 데이터 탐색의 신뢰도를 높였습니다.

01

사용자 입력 분석

활용 주제, 목적, 내용을 바탕으로 AI가 주요 키워드를 추출합니다.

02

공공데이터 후보 탐색

추출된 키워드를 공공데이터포털 목록조회 API에 전달하여 실제 데이터 후보를 조회합니다.

03

AI 후보 데이터 추천

조회된 데이터 목록 안에서만 AI가 적합한 데이터를 평가하고 추천합니다.

04

분석 아이디어 제공

추천 데이터의 활용 방향, 분석 아이디어, 시각화 방향을 함께 제공합니다.
